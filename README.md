# Zhongwen_xuexi — Nen tang hoc tieng Trung

> 每天学习，每天进步。 (Moi ngay hoc tap, moi ngay tien bo)

Website hoc tieng Trung co he thong tai khoan, kho Ebook HSK, hoc tu vung,
kiem tra kien thuc va tao doan van on tap bang AI (Gemini).

---

## Cong nghe su dung

- Frontend: React + Vite + React Router
- Giao dien: Tailwind CSS (tong mau cam chu dao)
- Database + Auth + Luu file: Supabase
- AI tao doan van: Google Gemini (gemini-flash-latest)

---

## Cach cai dat & chay

### Buoc 1: Cai thu vien
    npm install

### Buoc 2: Dien cau hinh
Mo file .env va dien:
- VITE_GEMINI_API_KEY (da co san)
- VITE_SUPABASE_URL va VITE_SUPABASE_ANON_KEY (lay tu project Supabase)

### Buoc 3: Chay website
    npm run dev

Mo trinh duyet tai: http://localhost:5173
